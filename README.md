# mllearn

Материалы для подготовки к junior ML / DS / GenAI интервью.

- `ml_interview_base.md` — базовая теория и контрольные вопросы.
- `ml_practice_tasks.md` — практические задачи по темам из базы.
- `data/` — готовые синтетические датасеты для задач.
- `scripts/generate_practice_data.py` — воспроизводимая генерация данных.

Перегенерировать данные:

```bash
python3 scripts/generate_practice_data.py
```
